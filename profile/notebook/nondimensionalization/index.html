Nondimensionalization

  1. Pay attention to the variables in the equation to be transformed.
  2. For each variable, identify the reference quantity for each dimension and present the equation with the reference quantities explicitly stated. For example, $t \to t T_{0}$. Only add a tilde to the non-dimensionalized variables.
  3. Gather the coefficients of each term and set appropriate reference quantities to make the coefficients equal to 1. At this point, set the reference quantities as fixed constants. Do not set values that can take a wide range and serve as good parameters, other than variables, as reference quantities. (In other words, use physical constants that are fixed from the beginning or constants that do not need to be changed as reference quantities.)
  4. Add a mark, such as a tilde, to the transformed variables.
  5. For fixed values other than the reference quantities, replace them if possible. However, in this case, do not add a tilde or other marks. Distinguish them from variables by using different characters or other appropriate measures.
